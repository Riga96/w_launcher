단축어 동작 원리

Safari에서 웹툰 페이지를 열고 공유 버튼을 누르면, 단축어가 현재 URL을 아래 주소로 보냅니다.

이 파일을 웹서버에서 열면 주소가 표시됩니다
설정 방법
  1. iPhone 단축어 앱 열기 → 오른쪽 상단 +
  2. 동작 추가 → Safari에서 열기 선택
  3. URL 입력란에 위 주소 복사 후 뒤에 단축어 입력 변수 추가
  4. 단축어 이름을 웹툰 기록으로 저장
  5. Safari 공유 시트에서 이 단축어를 탭하면 자동 저장
기존 작품이면 회차만 업데이트되고, 신규 작품이면 양식에 자동으로 채워집니다.
